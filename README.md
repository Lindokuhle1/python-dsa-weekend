# Python DSA Weekend Tasks

This repo contains beginner-friendly Python exercises for **Data Structures & Algorithms (DSA)**.

## Tasks

### Task 1 – Simple Telemetry Processor
- Compute maximum, minimum, and average voltage from a list of telemetry readings.
- List all voltages above the average.

### Task 2 – Reverse and Palindrome Checker
- Reverse a string representing a battery ID.
- Check if the string is a palindrome.

### Task 3 – Simple Stack Implementation
- Implement a stack using Python's list.
- Support `push`, `pop`, `peek`, and `is_empty`.
